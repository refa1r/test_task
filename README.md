# Тестовое задание: Superhero API
Данный проект представляет из себя решение тестовой задачи с использованием API супергероев

## Описание
Основные функции проекта:

Получает список всех супергероев из публичного API.

Фильтрует их по двум критериям:

gender — пол героя (male или female)

has_job — наличие работы (True или False)

Находит самого высокого героя в отфильтрованной выборке.

Проверяет корректность работы функций с помощью pytest.


Проект содержит несколько функций:

filter_heroes(data, gender, has_job)

Фильтрует список героев по параметрам:

gender — строка (учитывается в любом регистре)

has_job — булево значение
Возвращает список подходящих героев или пустой список
